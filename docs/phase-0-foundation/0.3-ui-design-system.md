Step 0.3 - UI/UX Design System
Phase: 0 - Foundation & Planning
Duration: 1-2 hours
Status: In Progress

Design Philosophy
Waylo Design Principles:

Budget-First - Always show price info prominently
Trust & Safety - Clear verification badges, real-time updates
Accessible - Large touch targets, high contrast, readable fonts
Minimal Friction - Quick actions, offline-first, no unnecessary steps
Community-Driven - Show user contributions, upvotes, badges


Color Palette

Primary Blue:    #0077BE  (Trust, navigation, reliability)
Primary Teal:    #00A896  (Adventure, freshness, eco-friendly)
Accent Colors
Accent Orange:   #FF6B35  (Energy, warnings, highlights)

Typography
Font Families
Primary Font: System Default

iOS: San Francisco
Android: Roboto
Web: -apple-system, BlinkMacSystemFont, "Segoe UI"

Why System Fonts?

Faster loading (no downloads)
Native feel on each platform
Excellent readability

Font Sizes & Usage
Hero Text:       32px / 2rem    (Page titles, branding)
H1:              28px / 1.75rem (Section headers)
H2:              24px / 1.5rem  (Card titles)
H3:              20px / 1.25rem (Subsections)
Body Large:      18px / 1.125rem (Important info, CTAs)
Body:            16px / 1rem    (Main content)
Body Small:      14px / 0.875rem (Secondary info)
Caption:         12px / 0.75rem (Timestamps, helper text)
Font Weights
Bold:            700  (Headers, emphasis)
Semibold:        600  (Subheaders, buttons)
Regular:         400  (Body text)
Light:           300  (De-emphasized text)

Spacing System
Using 8px base unit for consistent spacing:
xs:   4px   (Tight spacing, icon padding)
sm:   8px   (Between related items)
md:   16px  (Standard spacing)
lg:   24px  (Section spacing)
xl:   32px  (Major section breaks)
2xl:  48px  (Page padding)

Interaction Patterns
Gestures

Pull to Refresh - Update location list, reviews
Swipe - Navigate between photos, delete items
Long Press - Save location, quick actions
Pinch/Zoom - Map navigation
Double Tap - Like/upvote quickly

Loading States
Skeleton Screens:
- Show gray boxes where content will appear
- Subtle shimmer animation
- Better than spinners for content-heavy screens

Spinners:
- Use for short waits (< 2 seconds)
- Primary Teal color
- Center of screen or inline
Empty States
No Locations Found:
- 🗺️ Large icon
- "No locations nearby"
- "Be the first to add one!" 
- [Add Location] button

No Reviews Yet:
- ⭐ Large icon  
- "No reviews yet"
- "Share your experience!"
- [Write Review] button
Error States
Network Error:
- 📡 Icon
- "Connection lost"
- "Check your internet and try again"
- [Retry] button

Location Error:
- 📍 Icon
- "Can't access location"
- "Enable location services to find nearby spots"
- [Open Settings] button

Accessibility Guidelines
Touch Targets

Minimum size: 44x44px (iOS standard)
Spacing between: 8px minimum
Important actions: 48x48px or larger

Color Contrast

Text on white: 4.5:1 ratio minimum
Large text (18px+): 3:1 ratio minimum
Always test with accessibility tools

Text

Minimum font size: 16px for body text
Support dynamic type sizing
Clear, simple language
Avoid jargon

Images

All images have alt text
Icons have labels
Photos have descriptions for screen readers


Animation Guidelines
Timing
Fast:    150ms  (Hover, small changes)
Medium:  300ms  (Screen transitions, modals)
Slow:    500ms  (Large movements, page loads)
Easing
Standard:     ease-in-out  (Most transitions)
Enter:        ease-out     (Elements appearing)
Exit:         ease-in      (Elements disappearing)
Spring:       Custom       (Playful interactions)
Common Animations

Button Press - Scale down 0.95x, 100ms
Card Tap - Lift shadow, 150ms
Modal Open - Slide up + fade, 300ms
Navigation - Slide left/right, 300ms
Success - Checkmark draw + bounce, 500ms
